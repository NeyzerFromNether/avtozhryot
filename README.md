Описание (About)
АвтоЖрёт — веб-сервис для расчёта реальной стоимости владения автомобилем. Переводит расходы на топливо, страховку, обслуживание, налоги и износ в понятную метрику — ₽/км, с возможностью сравнить себя с другими водителями.

Демо-версия (static prototype): расчёты и данные хранятся в localStorage браузера. Бэкенд не подключён.

Что умеет

Главная страница (index.html)
Демо-калькулятор — показывает интерфейс без сохранения данных
Анимированная витрина сравнения (вы / средний / экономный сегмент)
Пошаговая инструкция «Как это работает»
FAQ, блок «О сервисе», декоративная SVG-машина в hero
Регистрация и вход через модальные окна (демо-аккаунт: demo@avtozhryot.ru / 123456)

Сервис (app.html)
Калькулятор стоимости владения: VIN/госномер, пробег, расход → ₽/км, ₽/мес, ₽/год, рейтинг 0–100
Инфографика: stacked bar + горизонтальные бары по статьям расходов
Прогноз расходов на 12 месяцев (Canvas)
Сравнительная таблица сегментов с анимированными полосами
Личный кабинет (появляется после входа):
Статистика: количество авто, расчётов, среднее ₽/км, рейтинг
Быстрый расчёт без перехода
Список автомобилей (до 5)
История расчётов с сортировкой (по дате, цене, рейтингу)
«А что если…?» — слайдеры пробега и расхода с пересчётом на лету
Сравнение с друзьями (демо-ранжирование)
Локальный топ (~120 профилей)
Советы по экономии с раскрывающимися карточками
FAQ

Темы и UX:
Тёмная / светлая тема с переключателем в шапке (☀️ / 🌙)
Сохранение выбора темы в localStorage, уважение системной prefers-color-scheme
Плавные переходы при переключении темы (0.35s)
Все цвета через CSS-переменные — ни одного хардкода фона/границы
Reveal-анимации при скролле (IntersectionObserver)
Полностью адаптивный дизайн (mobile-first)
Поддержка accessibility: ARIA-атрибуты, skip-link, семантическая разметка

Стек:
HTML5 + CSS3 (CSS Variables, Flexbox, Grid, Custom Properties)
Vanilla JavaScript (ES5/ES6+, без фреймворков)
Google Fonts: Inter + Montserrat
localStorage для хранения сессий, расчётов, авто и настроек темы
Canvas API для графика прогноза

Запуск
Откройте index.html в браузере. Никаких сборок, серверов и зависимостей не нужно. Для корректной работы localStorage откройте через http:// (не file://) или через Live Server / any static server.

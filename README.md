# ⛽ АвтоЖрёт

**Калькулятор реальной стоимости владения автомобилем.** Переводит расходы на топливо, страховку, обслуживание и налоги в понятную метрику — **₽/км**, с возможностью сравнить себя с другими водителями.

> Демо-версия (static prototype). Расчёты и данные хранятся в `localStorage` браузера. Бэкенд не подключён.

---

## 🚀 Быстрый старт

Откройте `index.html` в браузере. Никаких сборок и зависимостей.

**Демо-аккаунт:** `demo@avtozhryot.ru` / `123456`

> ⚠️ Для корректной работы `localStorage` откройте через `http://` (не `file://`) — используйте [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) в VS Code или любой static-сервер.

## ✨ Возможности

### Главная страница (`index.html`)

| Возможность | Описание |
|---|---|
| **Демо-калькулятор** | Показывает интерфейс целиком — без сохранения данных |
| **Анимированное сравнение** | Столбцы «Вы / Среднее / Экономный» с CSS-анимацией |

### Сервис (`app.html`)

| Возможность | Описание |
|---|---|
| **Калькулятор** | VIN/госномер, пробег, расход → ₽/км, ₽/мес, ₽/год, рейтинг 0–100 |

| **Советы по экономии** | Раскрывающиеся карточки |
| **FAQ** | Аккордеон |

### Личный кабинет (после входа)

| Возможность | Описание |
|---|---|
| **Статистика** | Количество авто, расчётов, среднее ₽/км, рейтинг |
| **Быстрый расчёт** | Калькулятор прямо в кабинете |
| **Мои автомобили** | До 5 авто, добавление по VIN / госномеру |
| **История расчётов** | Таблица с сортировкой (дата, цена, рейтинг) |
| **«А что если…?»** | Слайдеры пробега и расхода — пересчёт на лету |

| **Локальный топ** | Рейтинг из ~120 профилей |

## 🎨 Дизайн и UX

- **Тёмная / светлая тема** — переключатель в шапке (☀️ / 🌙), выбор сохраняется в `localStorage`, уважает системную `prefers-color-scheme`
- **Плавные переходы** — 0.35s при переключении темы
- **CSS-переменные** — все цвета управляются через кастомные свойства, ни одного хардкода фона или границы
- **Reveal-анимации** — элементы появляются при скролле (IntersectionObserver)
- **Mobile-first** — адаптивный дизайн для всех экранов
- **Accessibility** — ARIA-атрибуты, `skip-link`, семантическая разметка, фокус-стили

## 📌 Технические заметки

- **Бэкенд отсутствует** — всё работает на клиенте через `localStorage`
- В продакшене: замените `calculateOwnershipDemo()` в `app.js` на `fetch('/api/calculate', ...)`
- Хеширование паролей не реализовано — демо-логика; в продакшене нужна серверная аутентификация
- SVG-машина в hero-секции — декоративная, без внешних зависимостей

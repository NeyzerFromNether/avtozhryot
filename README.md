# avtozhryot
АвтоЖрёт — веб-сервис для расчёта реальной стоимости владения автомобилем. Переводит расходы на топливо, страховку, обслуживание, налоги и износ в понятную метрику — ₽/км, с возможностью сравнить себя с другими водителями.  Демо-версия (static prototype): расчёты и данные хранятся в localStorage браузера. Бэкенд не подключён.
